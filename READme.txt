Unzip the archive downloaded from the assigment. 
Right click on the unziped main folder and choose Open with Visual Studio Code
Once opended in VSCode click on the main folder and the go to menu ... -> Run->Run Without Debuging-> Choose browser et Voila!
You need to have Prettier, Live Server.
If issues appear try opening terminal in VSCode and navigate to the folder where the project is saved using cd commmand 
then run this command: npm init -y
And then npm install webpack webpack-cli --save-dev
